# Notebooks Index

This directory contains the primary analytical pipelines and exploratory data analysis for the Geolocate project.

## Main Pipeline

- **[`geolocate.ipynb`](geolocate.ipynb)**: ⭐ The central scientific contribution. Implements the full Bayesian Spatial AI framework (BYM2 model) for quantifying business survival probability across Brazilian municipalities.

## Exploratory & Supplementary

- **[`eda.ipynb`](eda.ipynb)**: Initial Exploratory Data Analysis covering municipality variables, spatial distributions, and correlation matrices.

## Archive (Gitignored)

The following notebooks are preserved for historical reference but are not part of the active pipeline:
- `bayesian_old.ipynb`: Legacy Bayesian implementation.
- `oldest.ipynb`: Early-stage prototyping.

## Installation

To run these notebooks, install the specific dependency stack:

```bash
pip install -r requirements_geolocate.txt
```
